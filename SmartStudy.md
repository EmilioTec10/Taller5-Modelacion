# SmartStudy

## Nombre del sistema
SmartStudy – Asistente Inteligente de Estudio

## Descripción general
SmartStudy es un sistema interactivo diseñado para ayudar a los estudiantes a organizar sus actividades académicas y mejorar sus hábitos de estudio. El sistema permite registrar tareas, recordatorios y horarios de estudio, además de enviar notificaciones cuando se acerca una fecha importante. También puede mostrar recomendaciones para optimizar el tiempo y mejorar la productividad del usuario.

Este sistema busca facilitar la planificación académica y reducir el estrés relacionado con la entrega de trabajos y evaluaciones.

## Usuario objetivo

El sistema está dirigido principalmente a estudiantes de secundaria y universitarios que necesitan organizar sus tareas, proyectos y horarios de estudio de manera más eficiente. También puede ser útil para cualquier persona que desee mejorar su planificación y gestión del tiempo.

## Entradas

El sistema SmartStudy recibe las siguientes entradas:

- *Entrada manual del usuario:*  
  El usuario puede ingresar tareas, fechas de entrega, horarios de estudio y recordatorios mediante la interfaz del sistema.

- *Fecha y hora del sistema:*  
  El sistema utiliza la fecha y hora del dispositivo para programar recordatorios y notificaciones.

- *Botones o interfaz táctil:*  
  Permiten al usuario interactuar con el sistema para agregar, modificar o eliminar información.

- *Sensor (opcional):*  
  Sensor de presencia o actividad del dispositivo (como detección de uso del computador o teléfono) para identificar cuándo el usuario está activo y enviar recordatorios en momentos oportunos.

- *Tipo de señal:*  
  Señal digital proveniente de la interfaz del usuario y del sensor del dispositivo.

## Salidas

El sistema SmartStudy genera las siguientes salidas:

- *Notificaciones al usuario:*  
  El sistema envía recordatorios cuando se acerca la fecha de entrega de tareas o cuando es momento de iniciar una sesión de estudio.

- *Visualización en pantalla:*  
  Muestra las tareas pendientes, horarios de estudio y recomendaciones de organización dentro de la interfaz del sistema.

- *Alertas sonoras (opcional):*  
  El sistema puede emitir sonidos para llamar la atención del usuario cuando haya recordatorios importantes.

- *Indicadores visuales (opcional):*  
  Cambios de color o mensajes emergentes que indiquen tareas urgentes o próximas a vencer.

- *Actuador (si aplica):*  
  Pantalla del dispositivo o altavoz del sistema.

- *Tipo de señal:*  
  Señales digitales enviadas a la pantalla del dispositivo y al altavoz para mostrar información o emitir alertas.
